## My Projects

### 2026 - Computational Intelligence for Optimisation - Girl with a Pearl Earring (MDSAA)
- Project currently in development, where we are using a Genetic Algorithm to approximate Vermeer's "Girl with a Pearl Earring" by evolving a set of 100 triangles rendered onto a 300x400 canvas.
- Implementing core GA components from scratch, including selection, crossover, and mutation operators, with fitness evaluated via RMSE against the target image.
- Built in Python using Pillow for rendering and a modular codebase split across dedicated components for population management, chromosome representation, and image generation.
- Code will be available soon [here](https://github.com/JohnnyR2003/CIFO-NOVAIMS2026)!

### 2026 - Deep Learning Project - Artwork Classification (MDSAA)
- A deep learning pipeline for image classification using a modified subset of the WikiArt dataset, consisting of scanned paintings from various authors.
- Explored multiple architectures, including custom CNNs, transfer learning with EfficientNetV2S and Xception, and state-of-the-art feature extraction with DINOv2.
- Built entirely with the Keras framework, following industry practices for generalisable model development.
- Check out the code [here](https://github.com/LuizaSalum/Deep-Learning-2026-MDSAA)!

### 2023 - Computation III Final Project - Car Racer Lolly Locket's Dog Chase (Information Systems)
